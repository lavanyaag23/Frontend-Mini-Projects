# ⚛️ Prop — React Mini Project

> A clean, minimal and component-based React application built using Create React App.

---

## ✨ Overview

This project is a simple React frontend application focused on learning component structure, state management, and reusable UI design.

---

## 🚀 Key Highlights

- ⚛️ Built with React functional components  
- 🧠 Uses state & props effectively  
- 🎨 Clean and responsive UI structure  
- ⚡ Fast setup with Create React App  
- 🧩 Modular and reusable components  

---

## 🛠️ Tech Stack

React.js • JavaScript (ES6+) • HTML5 • CSS3 • CRA
