# ⭐ Star Rating System

An interactive Star Rating System built using HTML, CSS, and JavaScript. Users can rate from 1 to 5 stars and receive instant visual and emoji feedback based on their selection.

## 🚀 Features

* ⭐ Interactive 5-Star Rating
* 😊 Dynamic Emoji Reactions
* 📊 Real-Time Rating Display
* 🔄 Reset Rating Functionality
* 🎨 Clean and User-Friendly Interface
* 📱 Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
