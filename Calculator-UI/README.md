# 🧮 Calculator UI

A simple and responsive calculator interface built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates the fundamentals of frontend development by creating a clean and interactive calculator capable of performing basic arithmetic operations.

## 🚀 Features

* Addition, Subtraction, Multiplication, and Division
* User-friendly interface
* Responsive design
* Interactive button layout
* Clear and Reset functionality
* Real-time calculation display

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
