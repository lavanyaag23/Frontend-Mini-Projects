# 🤖 Lavanya AI

Lavanya AI is a simple frontend chatbot built using **HTML, CSS, and JavaScript**. It acts as a friendly tech companion that can greet users, answer basic programming-related questions, share frontend development tips, and occasionally crack developer jokes.

The project was created to practice JavaScript, DOM manipulation, and chatbot interaction design while building an engaging user experience.

---

## 🚀 Features

* Interactive chat interface
* Friendly greetings and conversations
* Programming-related responses
* Frontend development guidance
* React.js suggestions and tips
* Developer humor and tech jokes
* Clean and responsive UI
* Sidebar chat history layout

---
