# 🌐 FutureTech Solutions Website

A simple multi-page responsive website built using **HTML5** and **CSS3**. This project represents a fictional IT company named **FutureTech Solutions** and demonstrates the fundamentals of web development, including page navigation, layouts, styling, and content organization.

## 🚀 Features

* Multi-page website structure
* Responsive navigation bar
* Home Page
* About Us Page
* Services Page
* Contact Us Page
* Clean and professional UI
* Consistent styling across all pages

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```text
WebPage/
│
├── index.html
├── about.html
├── services.html
├── contact.html
└── style.css
```

## 📄 Pages Included

### 🏠 Home Page

Introduces FutureTech Solutions and highlights the company's mission, values, and services.

### ℹ️ About Us

Provides information about the company, vision, goals, and team.

### 💼 Services

Displays the various technology and digital solutions offered by the company.

### 📞 Contact Us

Contains contact information and communication details for customers.
